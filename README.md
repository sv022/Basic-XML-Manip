# Basic-XML-Manip
Designed to manipulate xml files with the following structure:
```
<people>
  <person>
    <name></name>
    <birthdate></birthdate>
    <jobs>
      <job>
        <position></position>
        <start></start>
        <end></end>
        <department></department>
      </job>
    </jobs>
    <salaries>
      <salary>
        <year></year>
        <month></month>
        <total></total>
      </salary>
    </salaries>
  </person> 
</people>
```
Files are read from `bin/Debug/net7.0/test.xml`

Includes a basic command line interface.
